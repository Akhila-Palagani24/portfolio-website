# 🧠 Development Process

## Step 1 — Planning

The portfolio structure was planned with sections such as:

- Home
- About
- Skills
- Projects
- Contact

This created smooth navigation and better user experience.

---

## Step 2 — HTML Structure

HTML was used to build:

- Navigation bar
- Hero section
- Skills cards
- Projects section
- Contact section

Semantic structure was maintained for readability.

---

## Step 3 — Styling with CSS

CSS was used for:

- Responsive layouts
- Dark modern theme
- Hover effects
- Flexbox & Grid layouts
- Mobile responsiveness
- Smooth alignment and spacing

---

## Step 4 — Deployment

The website was deployed using Vercel for fast and reliable hosting.
